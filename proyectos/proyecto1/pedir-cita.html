Pedir Nueva Cita

Formulario para que los pacientes puedan programar una nueva cita médica.

Descripción

Los pacientes pueden seleccionar un médico asignado, elegir una fecha y hora disponible, especificar el motivo de la consulta y el tipo de cita (presencial o telefónica).

Capturas de pantalla

Formulario donde el paciente selecciona los detalles de la cita.

Rutas relacionadas

  • Frontend: `/pedir-cita` (Angular Router)
  • Backend: `/api/citas` (Spring Boot)

Flujo

  • Seleccionar médico → consultar disponibilidad → elegir fecha/hora → confirmar
  • Se crea la cita PROGRAMADA y se notifica por email

Validaciones

  • Fecha futura, hueco libre, motivo con 5–280 caracteres
  • Sin solapes con otras citas del paciente

API y errores

  • GET /api/citas/disponibilidad?medicoId&fecha
  • POST /api/citas → 201 cita | 409 solape | 422 datos inválidos

Ejemplo JSON

{
  "pacienteId": "p1",
  "medicoId": "m1",
  "fecha": "2025-05-20",
  "hora": "10:30",
  "tipoCita": "PRESENCIAL",
  "motivo": "Dolor de cabeza"
}
