App de Citas Médicas

Gestión de citas médicas para pacientes y médicos.

Introducción

Esta aplicación permite a pacientes y médicos gestionar citas, documentos médicos, medicación e información personal de forma sencilla y eficiente.

Tecnologías utilizadas

  • Frontend: Angular, Angular Material, Tailwind CSS
  • Backend: Spring Boot, Spring Security, JWT
  • Base de datos: MongoDB
  • Control de versiones: GitHub

Explorar funcionalidades

  • Gestionar Citas

  • Medicación
